# 🤖 Robocode Intelligent Systems Project

This repository contains the development of an autonomous battle robot for the game **Robocode**, created as part of the **Intelligent Systems** course at **Universidade Estadual do Sudoeste da Bahia (UESB)**.

The project focuses on applying concepts of Artificial Intelligence and intelligent decision-making in a competitive simulated environment.

---

# 📚 Course Information

- **University:** Universidade Estadual do Sudoeste da Bahia (UESB)
- **Department:** Departamento de Ciências Exatas e Tecnológicas (DCET)
- **Course:** Intelligent Systems
- **Professor:** Geraldo Pereira Rocha Filho

---

# 🎯 Project Objective

The main objective of this project is to develop an intelligent robot capable of competing autonomously in Robocode battles.

The robot must be able to:

- Detect enemies
- Move strategically
- Attack efficiently
- Avoid enemy attacks
- Make decisions based on environmental information
- Adapt its behavior according to battle conditions

The project also aims to explore Artificial Intelligence techniques applied to autonomous agents and strategic systems.

---

# 🧠 Intelligent Techniques

This project may include several Intelligent Systems approaches, such as:

- Rule-based systems
- Strategic behavior modeling
- Evasive movement algorithms
- Enemy tracking systems
- Predictive targeting
- Genetic algorithms (optional)
- Machine learning approaches (optional)

The robot's behavior is designed to improve combat efficiency and survivability during battles.

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/your-username/robocode-project.git
```

---

## 2. Open the Project

Open the project in your preferred Java IDE:

- IntelliJ IDEA
- VS Code
- Eclipse

---

## 3. Install Robocode

Download Robocode from the official website:

https://robocode.sourceforge.io

---

## 4. Add the Robot to Robocode

Copy the robot source code into the Robocode robots directory or configure the project path inside your IDE.

---

## 5. Compile and Run

Open Robocode and start battles against other robots.
